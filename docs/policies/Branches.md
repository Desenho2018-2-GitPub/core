# Criação de branches

* Para criar uma branch, o seguinte modelo deve ser aplicado:
```
TagSeExistir-NomeDaBranch
```

* Branch sem tag TAG:
```
NomeDaBranch
```

* Todos os espaços na **TAG** devem ser **"_"**:
```
Numero_da_issue-NomeDaBranch
```

* Os nomes das branches não deves ter espaços:
```
Numero_da_issue-NomeDaBranch
```

### Exemplo prático:
```
001-collision
```
